# دليل المطور

استخدم Python 3.11+ وأنشئ البيئة وفق README. نقطة الدخول `main.py`، ولا تنشئ `QApplication` ثانية. أي عمل قرص طويل يجب أن يبقى في services/worker ولا يحدّث QWidget مباشرة.

عند إضافة نهاية جديدة للعامل حافظ على قاعدة «إشارة terminal واحدة ثم worker_finished». لا تعرض حوارًا قبل `thread.finished`، ولا تستخدم lambda وسيطة لاستدعاء واجهة من إشارة عامل؛ اربط Slot تابعًا لـQObject في خيط الواجهة.

أضف لونًا جديدًا كرمز semantic في اللوحتين، ثم استخدمه في QPalette/QSS أو الرسم المخصص. لا تضف لونًا hex داخل صفحة. أضف زوج التباين إلى الاختبار عند استخدامه للنص.

قبل التسليم شغّل أوامر الجودة في README، ثم بنائي PyInstaller، ثم كل EXE مع `--smoke-test`. هذا الخيار التشخيصي ينفذ عمليتي نسخ متتاليتين، يغلق ملخصهما فقط، ويخرج 0 عند بقاء النافذة ودورة الخيط سليمتين.

ترحيلات SQLite تزايدية عبر `PRAGMA user_version`. خذ نسخة احتياطية قبل تعديل مخطط قائم ولا تحذف سجل المستخدم ضمن ترحيل.
