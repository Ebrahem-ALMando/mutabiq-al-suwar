# دليل مستخدم مُطابق الصور

ابدأ من «عملية جديدة». اسحب ملف XLSX/XLSM/CSV/TSV أو اختره، ثم تحقق من الورقة والعمود المقترح ونسبة الثقة. تظهر عينة قبل التنفيذ كي تتأكد من عدم سقوط الأصفار البادئة.

اختر مجلد الصور والوجهة. البحث العودي مناسب للبنى المتداخلة، لكن الفهرسة الأولى قد تستغرق وقتًا. لا تجعل المصدر والوجهة متطابقين.

المطابقة الدقيقة هي الأكثر أمانًا. التطبيع والقواعد تعمل بترتيبها؛ اختبرها قبل المتابعة. اقتراح fuzzy هو اقتراح مراجعة فقط. عند تعدد الصور اختر الأولى أو الكل، وعند تعارض الوجهة اختر التخطي أو الاستبدال أو إعادة التسمية.

شغّل المعاينة والفحص القبلي. الحالة «فشل» تمنع النسخ، و«تحذير» يسمح بالمتابعة بعد المراجعة. استخدم Dry Run لرؤية المسارات المتوقعة بلا كتابة ملفات.

بعد النسخ يظهر ملخص يمكن منه فتح الوجهة أو التقرير أو مراجعة المفقود. إغلاق الملخص لا يغلق التطبيق. من «السجل» افتح دفعة قديمة أو منشورها أو نفذ Undo الآمن.

لتغيير السمة أو حجم النص انتقل إلى «الإعدادات». تُطبق السمة فورًا على النوافذ والقوائم والحوارات المفتوحة. عند الإلغاء تُكمل عملية الملف الذرية الحالية ثم تتوقف قبل الملف التالي.

للتجربة، استخدم زر مساحة العرض من الصفحة الرئيسية؛ بيانات `samples/` خيالية ولا تلمس ملفاتك المعتادة.
