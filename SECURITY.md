# الأمان

التطبيق محلي ولا يرسل ملفات أو معرّفات عبر الشبكة. تتحقق قوالب الوجهة من منع traversal، وتُنقّى أسماء Windows المحظورة. النسخ يكتب ملفًا مؤقتًا داخل الوجهة ثم يزامنه ويتحقق من الحجم، ويمكن تفعيل SHA-256.

Undo مقيد بمنشور الدفعة وبحدود مجلد الوجهة، ويرفض حذف ملف تغير بعد النسخ. احتفظ بنسخ احتياطية مستقلة؛ Undo ليس نظام نسخ احتياطي ولا يعيد الملفات المستبدلة.

لا تفتح ملفات بيانات أو صورًا من مصدر غير موثوق بصلاحيات مرتفعة. أبلغ عن مشكلة أمنية مع الإصدار وخطوات إعادة الإنتاج ومقتطف سجل منزوع البيانات الشخصية، ولا ترفق بيانات فعلية.
