# سجل التغييرات

## 2.1.1 — 2026-07-20

- إصلاح شامل لقص أيقونات SVG عبر عرض متناسب مع `viewBox`، هامش آمن، تنعيم، وذاكرة مؤقتة واعية بـDPI.
- توحيد عناصر الشريط الجانبي في مكوّن RTL واحد كامل التفاعل مع حالات مستقلة للتحويم والنشاط والتركيز والتعطيل والطي.
- إصلاح ألوان الأيقونة النشطة وتباينها في السمتين، وتوحيد هندسة أزرار الأدوات العلوية.
- إضافة حقول رقمية قابلة لإعادة الاستخدام بهندسة ثابتة تفصل المحرر LTR عن أزرار الزيادة والنقصان، مع إصلاح القوائم والحقول المركبة.
- إضافة معرض المطوّر `--ui-audit` واختبارات انحدار ولقطات بصرية عند 1366×768 و1920×1080 وعوامل DPI متعددة.

## 2.1.0 — 2026-07-19

- تطبيق الهوية البصرية الرسمية بألوانها الفاتحة والداكنة وشعارها وأيقونة Windows الرسمية.
- نظام تصميم موحد بخط Tajawal وأيقونات SVG محلية ومبدّل سمة متحرك وحركة مخفّضة اختيارية.
- شريط جانبي RTL قابل للطي ومحفوظ الحالة، ولوحة بداية ومعالج من خمس خطوات بتصميم أكثر كثافة ووضوحاً.
- إصلاح معاينة النتائج model/view: صف Excel صريح، رأس عمودي مخفي، أعمدة وكثافة محفوظة، مرشحات وتصدير، وتبديل جدول/معرض.
- دليل تعليمي مصوّر داخل التطبيق مع اثني عشر رسماً محلياً وجولة تفاعلية اختيارية غير نمطية.
- حالات فارغة، حوارات وملخص رسمي، مناطق إسقاط محسّنة، وشريط تقدم متوافق مع السمتين.
- مصفوفة فحص بصري للوضعين عند 1366×768 و1920×1080، واختبارات هوية وأيقونات وRTL وحركة وجداول.

## 2.0.0 — 2026-07-19

- واجهة عربية متعددة الصفحات بسمتين وخط Tajawal مضمّن.
- معالج معاينة وفحص قبلي، CSV/TSV، قواعد مطابقة وقوالب وجهة آمنة.
- سجل SQLite وفهرس دائم ومنشورات JSON وUndo محافظ.
- نسخ ذري متحقق وتقارير ومحاكاة وملفات تعريف.
- إصلاح انهيار Windows بعد اكتمال النسخ عبر دورة حياة QThread محكومة وإشارات نهائية واضحة.
- نظام حوارات RTL وثيم semantic مركزي واختبارات تباين ودورة حياة GUI.
- حزم Windows one-folder وportable ومواصفات Inno Setup.
