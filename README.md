# مُطابق الصور

تطبيق مكتبي عربي يعمل محليًا على Windows لمطابقة المعرّفات الموجودة في Excel أو CSV/TSV مع أسماء الصور، ثم معاينة القرارات ونسخ الملفات بأمان مع تقرير وسجل قابل للتدقيق.

## أهم الإمكانات

- واجهة RTL بسمتين فاتحة وداكنة وخط Tajawal مضمّن، مع لوحة ألوان ذات تباين مُختبر.
- هوية رسمية مدمجة وشريط جانبي قابل للطي ومبدّل سمة متحرك وأيقونات SVG محلية بلا رموز Emoji.
- دليل مصوّر داخل التطبيق وجولة تفاعلية اختيارية، مع دعم تقليل الحركة.
- معالج من خمس مراحل: البيانات، المسارات، المطابقة، المعاينة والفحص القبلي، ثم التنفيذ.
- مطابقة دقيقة أو مطبّعة أو بقواعد بادئة/لاحقة وRegex، واقتراحات fuzzy لا تُنسخ دون قرار صريح.
- CSV/TSV مع اكتشاف الترميز والفاصل، وExcel بصيغتي XLSX وXLSM.
- نسخ ذري متحقق، وسياسات تخطي/استبدال/إعادة تسمية، وإعادة محاولة اختيارية وSHA-256.
- معاينة ومحاكاة Dry Run، وتقارير Excel، ومنشور JSON حصري لكل دفعة.
- سجل SQLite وفهرس صور دائم، وUndo محافظ لا يحذف إلا ملفًا أنشأته الدفعة ولم يتغير.
- تشغيل كامل بلا شبكة أو Telemetry؛ تبقى بيانات المستخدم على الجهاز.

## التثبيت والتشغيل

النسخ المبنية لا تحتاج إلى Python:

- مجلد التطبيق: `dist/MutabiqAlSuwar/MutabiqAlSuwar.exe`
- النسخة المحمولة: `dist/MutabiqAlSuwar-portable.exe`

للتطوير على Python 3.11 أو أحدث:

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt -r requirements-dev.txt
.\.venv\Scripts\python.exe main.py
```

## الاستخدام السريع

1. اختر ملف البيانات وورقة العمل وعمود المعرّف.
2. اختر مجلد الصور والوجهة، وحدد البحث داخل المجلدات الفرعية عند الحاجة.
3. اختر نمط المطابقة وسياسة التعارض، واختبر القواعد على أمثلة فعلية.
4. شغّل المعاينة؛ عالج النتائج المتعددة أو المقترحات اليدوية وراجع الفحص القبلي.
5. استخدم المحاكاة إن رغبت، ثم ابدأ النسخ. يبقى التطبيق مفتوحًا بعد الملخص ويمكن بدء دفعة أخرى.

أمثلة القوالب: `{identifier}_{sequence}.{extension}` لاسم الملف و`{excel_name}/{date}` للمجلد. تُنقّى الأسماء من محارف Windows المحظورة وتُرفض محاولات تجاوز مجلد الوجهة.

## البيانات المحلية

تُحفظ قاعدة البيانات والسجلات والمنشورات والملفات التعريفية تحت:

`%LOCALAPPDATA%\MutabiqAlSuwar`

الجداول الأساسية هي `batches` و`batch_items` و`indexed_folders` و`image_index` و`notifications`. تُطبّق ترحيلات تزايدية، وتؤخذ نسخة احتياطية قبل أي ترحيل لقاعدة قديمة.

## الاختبارات والجودة

```powershell
$env:QT_QPA_PLATFORM='offscreen'
.\.venv\Scripts\python.exe -m pytest -q
.\.venv\Scripts\python.exe -m ruff check . --exclude .venv --exclude .tools --exclude build --exclude dist
.\.venv\Scripts\python.exe -m black --check . --exclude "(\.venv|\.tools|build|dist)"
.\.venv\Scripts\python.exe -m mypy models repositories services utils workers
.\.venv\Scripts\python.exe scripts\benchmark.py
```

اختبارات الأداء الكبيرة اختيارية عبر `pytest -m performance`. لا تُضمّن fixtures ضخمة؛ تُنشأ مؤقتًا.

## بناء Windows

```powershell
.\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean mutabiq_onefolder.spec
.\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean mutabiq_onefile.spec
```

بعد تثبيت Inno Setup، افتح `installer/MutabiqAlSuwar.iss` أو نفّذ `ISCC.exe` عليه. المثبّت لا يحتاج صلاحيات مدير، ويعرض خيار اختصار سطح المكتب ويتيح الاحتفاظ ببيانات المستخدم أو حذفها عند الإزالة.

## بنية المشروع

- `models/`: نماذج الإعدادات والنتائج وحالة العملية.
- `services/`: القراءة والمطابقة والنسخ والتقرير والفحص والمنشور والتراجع.
- `repositories/`: SQLite والترحيلات والفهرس.
- `workers/`: عامل Qt وإشارات دورة الحياة.
- `ui/`: الصفحات والنماذج والحوارات والثيم.
- `assets/branding` و`assets/icons/lucide` و`assets/illustrations/guide`: الهوية والأيقونات والرسوم المحلية.
- `tests/`: اختبارات الوحدة والتكامل وpytest-qt.

## استكشاف المشكلات

- إذا لم تظهر مطابقة، راجع المعاينة والتطبيع وحالة الأحرف، ولا تفعّل قواعد واسعة بلا اختبار.
- في مجلدات الشبكة قد تكون الفهرسة والنسخ أبطأ وقد تتغير الصلاحيات أثناء الدفعة؛ استخدم الفحص القبلي وSHA-256 للبيانات الحساسة.
- إذا فشل التقرير بعد نجاح النسخ، تظهر النتيجة «جزئية» وتبقى الملفات المنسوخة محسوبة؛ راجع السجل ثم صدّر التقرير مجددًا.
- لا يُعد Undo نسخة احتياطية ولا يعيد ملفًا تم استبداله؛ يحذف فقط الملفات الجديدة غير المعدلة.

## القيود المعروفة

- لا تُنسخ اقتراحات fuzzy تلقائيًا.
- XLS القديم غير مدعوم؛ احفظه XLSX أو CSV.
- عرض المصغرات الكثيف يعتمد على سرعة القرص، بينما جدول النتائج نفسه يستخدم model/view.
- بناء Installer يتطلب تثبيت Inno Setup منفصلًا.

لقطات الشاشة غير مضمّنة في المستودع حاليًا؛ يمكن توليدها من مساحة العرض الخيالية في `samples/` من دون بيانات حقيقية.

## الترخيص والإقرارات

لم يُعلن لهذا المستودع ترخيص لإعادة توزيع شفرة التطبيق، لذلك لا يُفترض منح حقوق غير منصوص عليها. يعتمد التطبيق على Python وPySide6 وopenpyxl؛ راجع تراخيصها في توزيعاتها الرسمية. خط Tajawal من Google Fonts مضمّن وفق SIL Open Font License الموجودة في `assets/fonts/OFL.txt`. راجع `THIRD_PARTY_NOTICES.md`.
