# نسخ الصور حسب الرقم الذاتي

تطبيق مكتبي عربي لنظام Windows يقرأ المعرّفات من عمود في ملف Excel، ويفهرس مجلد الصور مرة واحدة، ثم يعاين التطابقات وينسخ الصور ذات اسم الملف المطابق تماماً إلى مجلد يحدده المستخدم. لا يعدّل التطبيق ملف Excel أو ملفات المصدر ولا يحتاج إلى تثبيت Microsoft Excel.

## الميزات الرئيسية

- واجهة عربية كاملة من اليمين إلى اليسار مبنية بـ PySide6.
- قراءة ملفات `.xlsx` و`.xlsm` واختيار ورقة العمل والعمود.
- اكتشاف تلقائي متسامح لعنوان العمود `الرقم الذاتي`، بما يشمل المسافات والتطويل وفواصل الأسطر.
- الحفاظ على النصوص والأصفار البادئة عند تخزينها كنص أو باستخدام تنسيق رقمي صريح مثل `000000`.
- مطابقة ساق اسم الملف كاملة فقط، مع خيار تجاهل حالة الأحرف.
- فهرسة المصدر مرة واحدة، مع بحث اختياري في المجلدات الفرعية واستبعاد الوجهة إن كانت داخل المصدر.
- دعم JPG وJPEG وPNG وWEBP وBMP وGIF وTIF وTIFF وHEIC وJFIF بأي حالة أحرف للامتداد.
- معاينة قبل النسخ، وإحصاءات فورية، وجدول قابل للبحث والفرز والتصفية.
- تشغيل القراءة والفهرسة والمطابقة والنسخ والتقرير داخل خيط خلفي كي تبقى الواجهة مستجيبة.
- إلغاء آمن بين الملفات ونسخ ذري عبر ملف مؤقت لتجنب ترك ملف وجهة ناقص.
- سياسات تخطي الملف الموجود أو استبداله أو إنشاء اسم متسلسل جديد.
- نسخ أول تطابق محدد ترتيبياً أو جميع الصور المطابقة.
- تقرير Excel احترافي بخمس أوراق وسجل تقني مفصل لكل عملية.
- حفظ الإعدادات الأخيرة وحجم النافذة وموضعها عبر `QSettings`.

## متطلبات التشغيل

- Windows 10 أو Windows 11.
- Python 3.11 أو أحدث.

من PowerShell داخل مجلد المشروع:

```powershell
py -3.11 -m venv .venv
Set-ExecutionPolicy -Scope Process Bypass
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python main.py
```

يمكن كذلك تشغيله مباشرة من دون تفعيل البيئة:

```powershell
.\.venv\Scripts\python.exe main.py
```

## تشغيل الاختبارات

ثبّت متطلبات التطوير ثم شغّل الاختبارات:

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements-dev.txt
.\.venv\Scripts\python.exe -m pytest -q
```

تعتمد الاختبارات على مجلدات وملفات مؤقتة ولا تحتاج إلى بيانات المستخدم.

## استخدام التطبيق

1. اختر ملف Excel بصيغة `xlsx` أو `xlsm`.
2. اختر ورقة العمل. يحاول التطبيق اختيار عمود `الرقم الذاتي` تلقائياً، ويمكن اختيار عمود آخر يدوياً.
3. اختر مجلد الصور ومجلد الحفظ، واضبط البحث الفرعي وسياسات التكرار.
4. استخدم **معاينة المطابقة** لمراجعة الإحصاءات والنتائج من دون نسخ.
5. استخدم **بدء المطابقة والنسخ** لتنفيذ النسخ وإنشاء التقرير.

المطابقة تتم على ساق اسم الملف كاملة. لذلك يطابق `a3222263` الملف `a3222263.jpg`، لكنه لا يطابق `new_a3222263.jpg` أو `a32222630.jpg`. لا تُحذف المسافات الداخلية أو الشرطات أو الشرطات السفلية. يمكن فقط إزالة المسافات المحيطة بالقيمة حسب الخيار المحدد.

## التكرار والتعارض

- **القيم المكررة في Excel:** تُتجاهل بعد ظهورها الأول افتراضياً مع الإبقاء عليها في النتائج والتقرير واحتساب عددها. يمكن تعطيل الخيار لمعالجتها مرة أخرى.
- **عدة صور للمعرّف نفسه:** ينسخ التطبيق أول مسار حسب ترتيب ثابت افتراضياً، أو ينسخ جميع التطابقات عند اختيار ذلك.
- **ملف موجود في الوجهة:** يمكن تخطيه، أو استبداله، أو إنشاء اسم جديد مثل `a3222263_2.jpg` ثم `_3` من دون استبدال أي ملف.

## التقرير والسجل

عند تمكين التقرير يُنشأ ملف باسم قريب من:

```text
تقرير_نسخ_الصور_2026-07-19_14-30-00.xlsx
```

ويحتوي أوراق `الملخص` و`جميع النتائج` و`غير الموجودة` و`الأخطاء` و`التكرارات`. تُنشأ السجلات التقنية في `logs/`، وتحتوي تفاصيل القرارات والأخطاء مع tracebacks لا تعرض في الواجهة.

## خط Tajawal

يحاول التطبيق تحميل الملفات التالية إن وُضعت في `assets/fonts/`:

- `Tajawal-Regular.ttf`
- `Tajawal-Medium.ttf`
- `Tajawal-Bold.ttf`

لا تُنزّل الخطوط تلقائياً لأسباب الترخيص والعمل دون اتصال. إن لم تكن موجودة يستخدم التطبيق `Segoe UI` أو `Arial` أو خط النظام من دون أن يتعطل.

## إنشاء ملف EXE مستقل

ضع ملفات الخط المرخصة في `assets/fonts/` إن رغبت بتضمينها، ثم نفّذ:

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements-dev.txt
.\.venv\Scripts\pyinstaller.exe --noconfirm --clean image_copy.spec
```

يظهر الملف التنفيذي في `dist\نسخ_الصور_حسب_الرقم_الذاتي.exe`. ملف `image_copy.spec` يضم مجلد `assets` كاملاً، وبالتالي يضم الخطوط عند وجودها. وللبناء دون ملف spec يمكن استخدام:

```powershell
.\.venv\Scripts\pyinstaller.exe --noconfirm --clean --windowed --name "نسخ_الصور_حسب_الرقم_الذاتي" --add-data "assets;assets" main.py
```

## بيانات التجربة

- `samples/sample_identifiers.xlsx`: ملف عمل عربي يحتوي عمود `الرقم الذاتي`.
- `samples/source_images/`: صور PNG صغيرة تطابق بعض القيم، مع قيمة غير موجودة وقيمة مكررة للتجربة.

يمكن إعادة إنشاء العينات في أي وقت:

```powershell
.\.venv\Scripts\python.exe scripts\create_sample_data.py
```

## استكشاف المشكلات

- **ملف Excel لا يفتح:** أغلقه مؤقتاً من البرامج الأخرى وتأكد من أنه ليس محمياً أو تالفاً وأن امتداده `xlsx` أو `xlsm`.
- **لم يُكتشف العمود:** اختره يدوياً من قائمة الأعمدة؛ يبحث التطبيق في أول 25 صفاً عن صف العناوين.
- **لا توجد صور مفهرسة:** تحقق من الامتدادات المدعومة، ومن خيار البحث في المجلدات الفرعية.
- **صورة لم تتطابق:** يجب أن تساوي القيمة ساق اسم الملف كاملة. راجع المسافات المحيطة وحالة الأحرف.
- **تعذر النسخ أو التقرير:** تحقق من صلاحية الكتابة والمساحة المتاحة، ثم راجع أحدث ملف داخل `logs/`.
- **مسارات عربية أو طويلة:** يستخدم التطبيق `pathlib` وواجهات Unicode في Python. قد تظل حدود المسار خاضعة لإعداد دعم المسارات الطويلة في Windows وللنظام والقرص المستخدمين.
