# خطة ترقية «مُطابق الصور»

> الحالة: نُفذت الخطة للإصدار 2.0.0. اجتازت النسخة النهائية 44 اختبارًا، مع اختبارين كبيرين اختياريين متجاوزين افتراضيًا، ونجح فحص دورة الحياة على نسختي Windows.

## خط الأساس

- التطبيق الحالي يفصل نماذج البيانات وخدمات Excel والفهرسة والنسخ والتقرير عن عامل Qt.
- المطابقة الدقيقة، سياسات التعارض، النسخ الذري، المعاينة، الإلغاء، التقرير والسجل تعمل، ومرّ خط الأساس بـ 19 اختباراً.
- نقاط الضعف: نافذة رئيسية كبيرة، `QTableWidget` غير مناسب لمئات آلاف الصفوف، غياب SQLite والسجل الدائم والمنشور ومرحلة الفحص القبلي والتراجع الآمن، وعدم وجود نظام سمات أو هيكل صفحات.

## مراحل التنفيذ

1. تأسيس الإصدار والمسارات والأخطاء وقاعدة البيانات والترحيلات والسجل والفهرس.
2. توسيع الإدخال إلى CSV/TSV، وإضافة التطبيع الصريح والأنماط والاقتراحات الضبابية والقوالب الآمنة.
3. إضافة الفحص القبلي والنسخ المتحقق مع retries والمنشور JSON والتراجع الآمن والملفات التعريفية.
4. استبدال واجهة النموذج بهيكل جانبي وصفحات ولوحة مؤشرات ومعالج خطوات ومعاينة model/view وسجل وإعدادات وحول.
5. إضافة السمات، السحب والإفلات، الإيقاف المؤقت، المحاكاة، الإشعارات، لوحة الأوامر وإتاحة الوصول.
6. توسيع الاختبارات والتوثيق، ثم lint/format/type/build وتشغيل حزمة Windows ومسار العرض الكامل.

يحافظ التحديث على مفاتيح `QSettings` الحالية ويضيف مفاتيح جديدة فقط. قاعدة البيانات تبدأ بمخطط ذي إصدار وتنفذ ترحيلات تزايدية دون حذف السجل.
